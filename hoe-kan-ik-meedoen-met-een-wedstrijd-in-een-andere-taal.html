Comment puis-je participer à un concours dans une autre langue?

Brandsupply est disponible en néerlandais, anglais, français, allemand et serbe.

Lorsque vous recherchez un concours sur cette page, vous pouvez indiquer dans quelles langues vous souhaitez effectuer votre recherche. Vous pouvez sélectionner jusqu'à cinq langues.

Peu importe d'où viennent les clients, nous leur recommandons toujours d'ajouter une description du concours en anglais.

De cette manière, ils atteignent le plus grand nombre possible de designers.

Il n'est pas possible d'ajouter une deuxième traduction (ou une troisième langue) à la description du concours.
